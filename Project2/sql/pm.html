PATTERN MATCHING

Pattern matching is checking whether a specific sequence of characters or token or data exist within a dataset. It is a primary feature of any data management language and is also implemented in SQL.
The LIKE operator is used with the WHERE clause.
It enables users to use _ to match a single character and % to match an arbitrary number of characters.
Pattern matching in MySQL is case-insensitive by default.
To make the pattern matching case sensitive in MySQL, the BINARY operator is used.

Syntax:
SELECT entry FROM data WHERE column LIKE 'pattern'; SELECT entry FROM data WHERE column NOT LIKE 'pattern';
